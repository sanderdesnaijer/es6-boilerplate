# ES6 Boilerplate

Simple es6 boilerplate with gulp tasks as:

  - browserify, watchify and babelify
  - jscs linter
  - sass

### Installation


```sh
$ npm install
```

##### Production build
For the minified versions of all files, run
```sh
$ gulp
```
##### Development
for development purpopes you can watch the files with
```sh
$ gulp --dev
```
